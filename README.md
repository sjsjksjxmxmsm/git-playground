# git-playground

一个专为学习 Git 操作和 Pull Request 流程设计的练习仓库。

## 目录

- [Git 教程](docs/Git教程.md) - 完整的 Git 学习指南
- [PR 提交指南](docs/PR提交指南.md) - 如何提交 Pull Request
- [个人简介](profiles/) - 贡献者介绍

## 使用方法

1. Fork 本仓库
2. 创建你的分支：`git checkout -b feature/your-name`
3. 在 `profiles/` 目录下添加你的个人简介
4. 提交并推送：`git push origin feature/your-name`
5. 创建 Pull Request
